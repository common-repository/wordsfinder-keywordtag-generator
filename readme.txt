Automatically generate tags/keywords via Wordsfinder API.

== Description ==

This plugin generates tags (keywords) for your posts while writing or editing, using the [Wordsfinder API](http://www.wordsfinder.com/ "Wordsfinder"). All the action happens in a box on the top right corner. Click on the generated tags to add them as categories or edit them as usual. Visit [Wordsfinder's home page](http://www.wordsfinder.com/ "Wordsfinder") for info on how to get a FREE API key.

== Installation ==

This section describes how to install the plugin and get it working.


1. Upload `wordsfinder` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Enter the plugin folder name (usually 'wordsfinder' or 'wordsfinder-keywordtag-generator')
4. Go to 'Wordsfinder' in the menu and enter your API key, set the options and click 'Update Options'

== Frequently Asked Questions ==

= The box stays on loading forever and I never get any tags =

Make sure you get an API key before using the Keyword generator.
